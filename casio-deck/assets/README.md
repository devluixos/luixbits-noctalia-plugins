# Assets

The first version uses Noctalia/Tabler glyphs:

- `device-watch`
- `device-watch-check`
- `device-watch-code`
- `device-watch-off`

If the plugin later needs a custom Casio-style icon, place the image here and
use `barWidget.setImage("assets/name.png", true, width, height)` from
`status.luau`.
